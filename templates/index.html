Renewable Energy (2015) - Course Project
DESCRIPTION
This project deals with the estimation of Power (max), Open-Circuit Voltage (Voc) and and Short-circuit Current (Isc) at any given location within US coordinates set, utilizing the available weather data parameters of solar radiation (kWh/m^2) and air temperate (in degree Celcius). The purpose of this course project is to demonstrate the ability to estimate the performance of a given solar panel at any given location within US, at any given time of the year.
DATA SOURCES USED
We utilized the estimated solar insolation and air temperature data for the years between 2005 to 2080. Also, we utilized three different models of solar panels, which are listed below:

  • YGE-U 72 CELL (Yingli Solar - Multi-crystalline)
  • Plus SW 280 mono (Sunmodule - Mono-crystalline)
  • KD325GX-LFB (Kyocera - Multicrystalline)
TECH USED
We built our project using following open-source technologies:

  • Python (data extraction + aggregation + backend architecture)
  • netCDF4 (data extraction)
  • Javascript (front-end + data querying mechanism)
  • Mongodb (data storage/retrieval)
